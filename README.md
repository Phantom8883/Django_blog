# Django Blog Project

Блог на Django из книги "Django 4 в примерах".

## Установка

```bash
pip install -r requirements.txt
python manage.py migrate
python manage.py createsuperuser
python manage.py runserver
```

## Функционал

- Посты блога с тегами
- Комментарии к постам
- Поиск постов
- Пагинация
- Кастомные шаблонные теги

## Технологии

- Django 4.1
- django-taggit
- Markdown
